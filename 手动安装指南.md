# 手动安装 PandaSU v1.4.49

如果一键安装脚本遇到问题，请使用以下手动方法：

## 方法一：直接复制安装（最简单）

### 步骤：
1. **找到 APK 文件**：
   ```
   c:\Users\Administrator\WorkBuddy\20260321183732\app\build\outputs\apk\debug\app-debug.apk
   ```

2. **复制到手机**：
   - 用 USB 线连接手机
   - 在电脑上打开手机存储
   - 将 `app-debug.apk` 复制到手机任意位置（如「下载」文件夹）

3. **在手机上安装**：
   - 用手机的文件管理器找到 APK 文件
   - 点击安装
   - 如果提示「未知来源」，点击「设置」→ 允许安装

## 方法二：使用微信文件传输

### 步骤：
1. **发送文件**：
   - 在电脑上打开微信
   - 将 `app-debug.apk` 发送到「文件传输助手」

2. **在手机上安装**：
   - 在手机上打开微信
   - 在「文件传输助手」中下载 APK 文件
   - 点击安装

## 方法三：使用 QQ/其他工具

### 步骤：
1. **发送文件**：
   - 通过 QQ、钉钉、云存储等工具发送 APK 到手机

2. **安装**：
   - 在手机上接收并安装

## 安装后验证

### 检查版本：
1. 打开 PandaSU
2. 进入「关于」或「设置」页面
3. 确认版本号为 **1.4.49**

### 测试功能：
1. **系统应用分类**：
   - 进入应用列表
   - 检查用户应用数量是否为约 **134 个**（不是 157 个）

2. **安装页面功能**：
   - 进入「安装」页面
   - 测试三个按钮的响应：
     - "选择一个文件"：应弹出文件选择器
     - "直接安装"：应有安装进度
     - "安装到未使用的槽位"：应有槽位检测

## 问题排查

### 问题：安装失败
**解决方案**：
1. 卸载旧版本 PandaSU
2. 重新安装新版本
3. 确保手机允许「未知来源安装」

### 问题：应用闪退
**解决方案**：
1. 清除应用数据
2. 重新安装
3. 重启手机

### 问题：功能异常
**解决方案**：
1. 检查手机是否已 Root
2. 检查是否安装了 KernelSU/Magisk
3. 查看应用内日志
