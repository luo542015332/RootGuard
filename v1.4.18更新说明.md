# PandaSU v1.4.18 更新说明

## 新增功能
**日志导出功能**：可以将 PandaSU 应用日志导出为文本文件，方便问题排查和分享。

## 功能详情

### 日志导出
- **导出位置**：应用内部存储目录（`/data/data/com.rootguard.app/files/`）
- **文件命名**：`PandaSU_Logs_YYYYMMDD_HHMMSS.txt`
- **导出内容**：
  - 导出时间和日志数量
  - 所有日志条目（带时间戳和日志级别）
  - 支持的日志级别：INFO、WARNING、ERROR、DEBUG

### 使用方法
1. 打开 PandaSU 应用
2. 进入"日志"页面
3. 点击右上角的 **导出按钮**（分享图标）
4. 确认导出操作
5. 日志将保存到应用内部存储
6. 使用文件管理器查看或分享该文件

## 版本信息
- **versionCode**: 32
- **versionName**: 1.4.18

## 技术细节
- 使用 `buildString` 高效构建日志内容
- UTF-8 编码确保中文正常显示
- 文件名包含时间戳避免覆盖
- 导出文件存储在应用私有目录，不需要额外权限
