- **IsolationScreen**: 额外检查包名前缀(com.miui., com.xiaomi.等)

### 现在的统一逻辑
所有地方都使用 `MagiskProvider.isSystemApp()`:
```kotlin
private fun isSystemApp(appInfo: ApplicationInfo): Boolean {
    // 方法1: 检查 FLAG_SYSTEM 标志
    if ((appInfo.flags and FLAG_SYSTEM) != 0) {
        return true
    }
    
    // 方法2: 检查安装路径
    val sourceDir = appInfo.sourceDir ?: return false
    val isSystemPath = sourceDir.contains("/system/") ||
                      sourceDir.contains("/vendor/") ||
                      sourceDir.contains("/product/") ||
                      sourceDir.contains("/apex/")
    
    return isSystemPath
}
```
