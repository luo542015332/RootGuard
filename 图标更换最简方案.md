# 🐼 图标更换 - 最简单方案

## 🎯 最简单的方法（推荐）

### 第一步：准备图标

1. 找到你的熊猫图标
2. 保存为 `panda.png`
3. **重要**：建议尺寸 **512×512** 或更大
4. 格式：PNG

---

### 第二步：使用在线工具

**访问这个网站：**
👉 https://www.appicon.co/

**操作步骤：**

1. 点击网站上的 **"Choose File"** 按钮
2. 选择你的 `panda.png`
3. 在 "Platform" 选项中选择 **"Android"**
4. 点击 **"Generate"** 按钮
5. 等待生成完成
6. 点击 **"Download"** 下载 ZIP 文件

---

### 第三步：解压并复制

1. **解压**下载的 ZIP 文件
2. **打开**解压后的文件夹
3. 找到所有 `.png` 文件（应该有很多不同尺寸的）
4. **复制**所有 PNG 文件到项目：

   ```
   项目位置：c:\Users\Administrator\WorkBuddy\20260321183732\app\src\main\res\

   复制到：
   ├── mipmap-mdpi/      → 所有 48×48 的 PNG
   ├── mipmap-hdpi/      → 所有 72×72 的 PNG
   ├── mipmap-xhdpi/     → 所有 96×96 的 PNG
   ├── mipmap-xxhdpi/   → 所有 144×144 的 PNG
   └── mipmap-xxxhdpi/  → 所有 192×192 的 PNG
   ```

5. **重命名**每个文件夹中的文件为：
   - `ic_launcher.png`
   - `ic_launcher_round.png`

---

### 第四步：构建和安装

完成复制后，运行以下命令：

```bash
# 清理缓存
.\gradlew.bat clean

# 构建 APK
.\gradlew.bat assembleDebug

# 安装到设备
.\platform-tools\adb.exe install app\build\outputs\apk\debug\app-debug.apk
```

或者使用 GitHub Actions：
1. 推送代码：`git add . && git commit -m "feat: 更新图标" && git push`
2. 等待 5-10 分钟
3. 从 GitHub Actions 页面下载 APK
4. 安装到设备

## 🔍 常见问题

### Q: 在线工具打不开？
**A:** 试试这些网站：
- https://www.appicon.co/
- https://romannurik.github.io/AndroidAssetStudio/icons-launcher.html
- https://makeicon.app/

### Q: 找不到对应尺寸的文件？
**A:**
- 在线工具通常会生成所有需要的尺寸
- 如果缺少某个尺寸，可以用画图工具手动调整
- 或者使用最近的大尺寸图标复制

### Q: 图标不显示？
**A:**
- 运行 `.\gradlew.bat clean` 清理缓存
- 确保文件名正确：`ic_launcher.png` 和 `ic_launcher_round.png`
- 重启设备后查看

### Q: 图标模糊？
**A:**
- 使用更高分辨率的原图（1024×1024）
- 确保生成的图标尺寸正确

### Q: 不想用在线工具？
**A:**
- 运行 `.\图标更换快速指南.bat`
- 选择方式二或方式三
