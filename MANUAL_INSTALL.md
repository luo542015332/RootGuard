# RootGuard 手动安装指南

## 📱 手机直接安装步骤

### 方法一：使用文件管理器安装（最简单）

#### 步骤 1：获取 APK 文件

**方式 A - 电脑传输**
1. 用 USB 线连接手机和电脑
2. 在手机上选择「文件传输」模式
3. 将 `app-debug.apk` 复制到手机存储（如 Download 文件夹）

**方式 B - 直接下载**
1. 在手机浏览器中访问 GitHub Releases
2. 下载 `app-debug.apk`

#### 步骤 2：允许未知来源安装

**Android 8.0+:**
1. 设置 → 应用 → 特殊应用权限 → 安装未知应用
2. 找到你的文件管理器（如「文件管理」、「ES文件浏览器」等）
3. 开启「允许来自此来源的应用」

**Android 7.0 及以下:**
1. 设置 → 安全
2. 开启「未知来源」

#### 步骤 3：安装 APK

1. 打开文件管理器
2. 找到 `app-debug.apk` 文件
3. 点击安装
4. 等待安装完成
5. 点击「打开」启动应用

---

### 方法二：使用第三方应用安装

#### 使用「发送到手机」功能

**Windows:**
1. 右键点击 APK 文件
2. 选择「发送到」→「蓝牙设备」（如果手机支持）
3. 或使用「就近共享」（Nearby Share）

**Mac:**
1. 右键点击 APK 文件
2. 选择「共享」→「隔空投送」（AirDrop）
3. 选择你的手机

#### 使用即时通讯工具

1. 将 APK 发送到微信/QQ/Telegram 文件传输助手
2. 在手机上打开聊天窗口
3. 点击下载 APK
4. 下载完成后点击安装

---

### 方法三：使用无线 ADB（无需 USB 线）

#### 前提条件
- 手机和电脑在同一 WiFi 网络
- 已开启 USB 调试（首次需要 USB 线）

#### 步骤

**1. 首次设置（需要 USB 线）**
```bash
# 连接手机（USB）
adb devices

# 启用网络调试
adb tcpip 5555

# 断开 USB，查看手机 IP
adb shell ip addr show wlan0
```

**2. 无线连接**
```bash
# 连接手机（无线）
adb connect 192.168.x.x:5555

# 验证连接
adb devices

# 安装 APK
adb install app-debug.apk
```

## 🔧 安装问题解决

### 问题 1：提示「应用未安装」

**原因：**
- APK 文件损坏
- 已安装签名不同的同名应用

**解决：**
1. 重新下载/复制 APK
2. 先卸载旧版本再安装

### 问题 2：提示「解析包时出现问题」

**原因：**
- APK 下载不完整
- Android 版本不兼容

**解决：**
1. 重新下载 APK
2. 确认 Android 版本 ≥ 8.0

### 问题 3：安装按钮灰色无法点击

**原因：**
- 未允许未知来源安装

**解决：**
1. 按上述步骤开启「允许未知来源」
2. 重新打开 APK 安装界面
