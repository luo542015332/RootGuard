# 🐼 使用你的熊猫图标作为应用图标

## 🎯 快速开始

1. **准备你的熊猫图标**
   - 建议尺寸：512×512 或更大
   - 格式：PNG（推荐，支持透明背景）
   - 文件名：icon.png（可选）

2. **使用在线工具**
   - 浏览器应该已经打开了：https://www.appicon.co/
   - 如果没有打开，请手动访问上述链接

3. **上传并生成图标**
   - 点击网站上的"上传"按钮
   - 选择你的熊猫图标
   - 选择 "Android" 平台
   - 点击"下载"获取图标包

4. **复制到项目**
   - 解压下载的 ZIP 文件
   - 将所有 PNG 文件复制到 `app/src/main/res/mipmap-*/` 对应的文件夹中

## 📁 需要的文件

每个 mipmap 文件夹需要两个文件：

| 文件夹 | ic_launcher.png | ic_launcher_round.png |
|--------|-----------------|-----------------------|
| mipmap-mdpi | 48×48 | 48×48 |
| mipmap-hdpi | 72×72 | 72×72 |
| mipmap-xhdpi | 96×96 | 96×96 |
| mipmap-xxhdpi | 144×144 | 144×144 |
| mipmap-xxxhdpi | 192×192 | 192×192 |

## ✅ 完成后的验证

复制完所有文件后，运行以下命令：

```bash
# 1. 清理构建缓存
.\gradlew.bat clean

# 2. 重新构建
.\gradlew.bat assembleDebug

# 3. 安装到设备
.\install-apk.ps1
```

然后在手机上查看应用图标，应该就能看到你的可爱熊猫了！

## 🆘 遇到问题？

- **图标没有更新？** → 运行 `.\gradlew.bat clean` 清理缓存
- **图标显示模糊？** → 检查原图分辨率是否足够（建议 512×512）
- **图标被裁剪？** → 确保图标内容居中，留出足够的边距
