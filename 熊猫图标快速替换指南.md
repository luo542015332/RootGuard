## 🚫 如果不想安装 ImageMagick

### 手动处理方案

如果不想安装 ImageMagick，可以使用在线工具：

**推荐工具：appicon.co**
1. 访问：https://appicon.co/
2. 上传你的 `panda.png`
3. 选择 Android 平台
4. 下载生成的图标包
5. 手动复制文件到对应的 mipmap 文件夹

**需要的文件列表**：

| 分辨率 | 路径 | 文件名 |
|--------|------|--------|
| 36x36 | `app/src/main/res/mipmap-mdpi/` | ic_launcher.png |
| 48x48 | `app/src/main/res/mipmap-hdpi/` | ic_launcher.png |
| 72x72 | `app/src/main/res/mipmap-xhdpi/` | ic_launcher.png |
| 96x96 | `app/src/main/res/mipmap-xxhdpi/` | ic_launcher.png |
| 144x144 | `app/src/main/res/mipmap-xxxhdpi/` | ic_launcher.png |

**圆形图标**（相同分辨率，文件名改为 `ic_launcher_round.png`）

## 🔄 更新图标配置

将所有尺寸的 PNG 文件放置到相应文件夹后，更新配置文件。

### 删除或修改 XML 配置文件

每个 mipmap 文件夹中有 `ic_launcher.xml` 和 `ic_launcher_round.xml`，需要：

**选择 A：删除 XML 文件**（推荐）
- 删除所有 `ic_launcher.xml` 和 `ic_launcher_round.xml`
- Android 会自动使用同名的 PNG 文件

**选择 B：修改 XML 引用**
- 将所有 XML 文件内容改为：
```xml
<?xml version="1.0" encoding="utf-8"?>
<bitmap xmlns:android="http://schemas.android.com/apk/res/android"
    android:src="@mipmap/ic_launcher" />
```
