# PandaSU 应用图标更新说明

## 📸 方法一：使用在线工具（推荐）

这是最简单的方法，只需要上传你的图片，自动生成所有尺寸。

### 步骤：

1. **访问在线图标生成工具**
   - 推荐工具：https://www.appicon.co/
   - 或者：https://romannurik.github.io/AndroidAssetStudio/icons-launcher.html

2. **上传你的熊猫图标**
   - 建议原图至少 512×512 像素
   - PNG、JPG 或 WebP 格式都可以
   - 建议使用透明背景的 PNG

3. **下载生成的图标包**
   - 选择 Android 适配
   - 下载后会得到一个包含所有尺寸的 ZIP 包

4. **解压并复制到项目**
   ```
   解压后的文件夹 -> app/src/main/res/mipmap-*/
   ```

---

## 🛠️ 方法二：使用自动脚本

如果已安装 ImageMagick，可以使用自动脚本：

### 步骤：

1. **安装 ImageMagick**
   - 下载：https://imagemagick.org/script/download.php
   - 安装时勾选 "Install legacy utilities (e.g. convert)"

2. **准备图标**
   - 将你的熊猫图标保存为 `icon.png` 放到项目根目录
   - 建议至少 512×512 像素

3. **运行脚本**
   ```powershell
   .\update-icon.ps1
   ```

4. **清理和重新构建**
   ```bash
   .\gradlew.bat clean
   ```

---

## 🎨 方法三：手动处理

如果需要精确控制图标效果，可以手动处理。

### 图标尺寸要求

| 文件夹 | 分辨率 (像素) | 用途 |
|--------|-------------|------|
| `app/src/main/res/mipmap-mdpi/` | 48×48 | 低密度屏幕 |
| `app/src/main/res/mipmap-hdpi/` | 72×72 | 中密度屏幕 |
| `app/src/main/res/mipmap-xhdpi/` | 96×96 | 高密度屏幕 |
| `app/src/main/res/mipmap-xxhdpi/` | 144×144 | 超高密度屏幕 |
| `app/src/main/res/mipmap-xxxhdpi/` | 192×192 | 超超高密度屏幕 |

### 文件命名

每个文件夹中需要两个文件：
- `ic_launcher.png` - 标准图标（方形）
- `ic_launcher_round.png` - 圆形图标（带圆形遮罩）

### 手动步骤

1. **运行占位符脚本**
   ```bash
   .\prepare-icon-folders.bat
   ```

2. **使用图片编辑工具缩放**
   - Photoshop、GIMP、Paint.NET 等
   - 或使用在线工具如 Fotor、Canva

3. **替换占位符文件**
   - 将生成的图标覆盖到对应文件夹

## ✅ 完成后的验证

完成图标替换后：

1. **刷新 Android Studio 项目**
   - 点击菜单：`File` -> `Sync Project with Gradle Files`

2. **清理构建缓存**
   ```bash
   .\gradlew.bat clean
   ```

3. **重新构建**
   ```bash
   .\gradlew.bat assembleDebug
   ```

4. **安装测试**
   ```bash
   .\install-apk.ps1
   ```

5. **查看效果**
   - 在手机上查看应用图标是否正确显示
   - 检查不同设备密度的显示效果

## 📌 注意事项

- ✅ **PNG 格式**：确保使用 PNG 格式，支持透明通道
- ✅ **清晰度**：保持图标在不同尺寸下的清晰度
- ✅ **居中显示**：圆形图标的内容要居中，避免被裁剪
- ✅ **尺寸一致**：标准图标和圆形图标尺寸要一致
- ⚠️ **不要删除 XML 文件**：mipmap 文件夹中的 XML 文件是必需的，不要删除

---

## 🔧 常见问题

### Q1: 图标没有更新怎么办？
**A:** 运行 `.\gradlew.bat clean` 清理缓存，然后重新构建

### Q2: 图标显示模糊怎么办？
**A:** 检查原图分辨率是否足够（建议至少 512×512）

### Q3: 圆形图标被裁剪了怎么办？
**A:** 确保图标内容居中，留出足够的边距

### Q4: 想用矢量图标怎么办？
**A:** 可以恢复 `ic_panda.xml` 的使用，更新 `mipmap-anydpi-v26` 配置即可
