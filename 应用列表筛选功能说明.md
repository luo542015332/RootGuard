# 应用列表筛选功能

## 功能概述

为 RootGuard 的应用管理页面添加了完整的应用筛选功能，允许用户：

1. **查看所有应用** - 显示系统应用和用户应用
2. **筛选系统应用** - 只显示系统预装应用
3. **筛选用户应用** - 只显示用户安装的应用
4. **按 Root 权限筛选** - 已授权、已拒绝、待询问
5. **应用统计** - 显示用户应用、系统应用和权限状态的数量

---

## 问题背景

**用户反馈**: "应用里只能看到系统应用，看不到自己装的"

**原因分析**:
- 原来的应用列表显示所有应用，但缺少按类型筛选的功能
- 没有视觉提示区分系统应用和用户应用
- 用户无法快速找到自己安装的应用

**解决方案**:
- 添加应用类型筛选选项（系统应用/用户应用）
- 优化筛选菜单 UI，使用图标和 emoji 增强视觉效果
- 添加统计卡片，显示不同类型应用的数量

2. **更新 applyFilters 方法**:
```kotlin
private fun applyFilters(apps: List<AppItem>, query: String, filter: AppFilter): List<AppItem> {
    var result = apps

    // 应用搜索过滤
    if (query.isNotBlank()) {
        result = result.filter {
            it.name.contains(query, ignoreCase = true) ||
            it.packageName.contains(query, ignoreCase = true)
        }
    }

    // 应用筛选
    result = when (filter) {
        AppFilter.ALL -> result
        AppFilter.SYSTEM -> result.filter { it.isSystemApp }        // 系统应用
        AppFilter.USER -> result.filter { !it.isSystemApp }        // 用户应用
        AppFilter.GRANTED -> result.filter { it.rootStatus == RootAccessStatus.GRANTED }
        AppFilter.DENIED -> result.filter { it.rootStatus == RootAccessStatus.DENIED }
        AppFilter.PROMPT -> result.filter { it.rootStatus == RootAccessStatus.PROMPT }
    }

    return result
}
```

3. **更新 updateApps 方法**:
```kotlin
private fun updateApps(apps: List<AppItem>) {
    val userApps = apps.count { !it.isSystemApp }
    val systemApps = apps.count { it.isSystemApp }
    val granted = apps.count { it.rootStatus == RootAccessStatus.GRANTED }
    val denied = apps.count { it.rootStatus == RootAccessStatus.DENIED }
    val prompt = apps.count { it.rootStatus == RootAccessStatus.PROMPT }

    _uiState.update { state ->
        val filtered = applyFilters(apps, state.searchQuery, state.filter)

        state.copy(
            apps = apps,
            filteredApps = filtered,
            userAppCount = userApps,
            systemAppCount = systemApps,
            grantedCount = granted,
            deniedCount = denied,
            promptCount = prompt,
            isLoading = false
        )
    }
}
```

## 用户使用方法

### 查看所有应用
1. 打开"应用授权"页面
2. 点击右上角的筛选图标（📊）
3. 选择"全部应用"

### 只看用户应用
1. 打开"应用授权"页面
2. 点击右上角的筛选图标（📊）
3. 选择"📱 用户应用"

### 只看系统应用
1. 打开"应用授权"页面
2. 点击右上角的筛选图标（📊）
3. 选择"⚙️ 系统应用"

### 按 Root 权限筛选
1. 打开"应用授权"页面
2. 点击右上角的筛选图标（📊）
3. 选择相应权限状态：
   - "✅ 已授权" - 只显示已授权的应用
   - "❌ 已拒绝" - 只显示已拒绝的应用
   - "❓ 待询问" - 只显示待询问的应用

### 搜索应用
1. 在搜索框输入应用名称或包名
2. 应用列表会实时过滤显示匹配结果
3. 搜索可以和筛选功能配合使用

## 数据来源

### 应用类型判断
系统应用判断逻辑（来自 MagiskProvider）：
```kotlin
val isSystemApp = (appInfo.flags and ApplicationInfo.FLAG_SYSTEM) != 0
```

**说明**:
- `FLAG_SYSTEM` 标志表示应用是系统应用
- 系统应用通常安装在 `/system/` 或 `/system/app/` 目录
- 用户应用安装在 `/data/app/` 目录
