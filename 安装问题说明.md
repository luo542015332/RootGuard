# 🔧 安装问题说明

## ⚠️ 当前问题

无法构建 APK，原因：

**Android SDK 不完整**

缺少的组件：
- `cmdline-tools` - SDK 命令行工具
- `platforms/android-34` - Android 34 平台
- `build-tools/34.0.0` - 构建工具 34.0.0

---

## ✅ 解决方案

### 方案一：使用 Android Studio（推荐）

1. **下载并安装 Android Studio**
   - 访问：https://developer.android.com/studio
   - 下载并安装 Android Studio

2. **安装必要的 SDK 组件**
   - 打开 Android Studio
   - 进入 `Tools` → `SDK Manager`
   - 安装以下组件：
     - ✅ `Android 14.0 (API 34)` - SDK Platform
     - ✅ `Android SDK Build-Tools 34`
     - ✅ `Android SDK Command-line Tools (latest)`

3. **接受许可协议**
   - 在 SDK Manager 中，勾选 "Accept License Agreements"

4. **构建项目**
   - 在 Android Studio 中打开项目
   - 选择 `Build` → `Make Project`
   - 或者 `Build` → `Generate Signed Bundle / APK`

---

### 方案二：手动安装 SDK 组件

1. **下载 Command Line Tools**
   - 访问：https://developer.android.com/studio#command-tools
   - 下载 "Command line tools only"

2. **解压并安装**
   ```
   解压到: C:\Android\Sdk\cmdline-tools\latest\
   ```

3. **使用 sdkmanager 安装组件**
   ```bash
   C:\Android\Sdk\cmdline-tools\latest\bin\sdkmanager.bat "platforms;android-34" "build-tools;34.0.0"
   ```

4. **接受许可**
   ```bash
   C:\Android\Sdk\cmdline-tools\latest\bin\sdkmanager.bat --licenses
   ```

---

### 方案三：使用在线构建服务（暂不可用）

如果不想在本地构建，可以使用：
