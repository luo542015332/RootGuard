## 安装方法

### 方法 1：手动安装（推荐）
1. 将 `app-debug.apk` 复制到手机存储
2. 在手机上使用文件管理器打开该文件
3. 允许"从未知来源安装"
4. 点击安装

### 方法 2：使用 ADB 安装
```bash
# 连接手机到电脑并开启 USB 调试
adb devices

# 安装 APK
adb install -r "c:\Users\Administrator\WorkBuddy\20260321183732\app\build\outputs\apk\debug\app-debug.apk"
```

### 方法 3：通过文件管理器
1. 将 APK 文件上传到云存储（如微信文件传输助手）
2. 在手机上下载并安装
