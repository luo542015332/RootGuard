# 🐼 手动设置熊猫图标详细步骤

由于无法在线上传，我们使用本地工具手动处理。

---

## 📋 准备工作

### 第一步：保存原图

1. 找到你的熊猫图标
2. 将其保存为 `panda.png`
3. 建议尺寸：**512×512** 或更大
4. 建议格式：**PNG**（支持透明背景）

---

## 🎨 使用 Windows 画图工具

### 第二步：调整图标尺寸

#### 方法 1：使用画图工具

1. **打开图片**
   - 右键点击 `panda.png`
   - 选择"打开方式" → "画图"

2. **调整大小**
   - 点击菜单"画图" → "调整大小"
   - 取消"保持纵横比"（如果需要）
   - 输入想要的尺寸（例如：48 x 48）
   - 点击"确定"

3. **保存副本**
   - 点击"文件" → "另存为"
   - 选择"PNG 图片"
   - 文件名：`panda_48.png`
   - 点击"保存"

4. **重复以上步骤**，分别保存：
   - `panda_48.png` (48×48)
   - `panda_72.png` (72×72)
   - `panda_96.png` (96×96)
   - `panda_144.png` (144×144)
   - `panda_192.png` (192×192)

#### 方法 2：使用在线工具

如果可以访问其他网站，试试这些：
- https://picresize.com/
- https://www.resizeimage.net/
- https://www.iloveimg.com/resize-image

## 🎯 快捷操作提示

### 批量复制命令（PowerShell）

如果你已经生成了所有尺寸的图标，可以使用以下脚本快速复制：

```powershell
# 将此脚本保存为 copy-icons.ps1 并运行

$projectPath = "app\src\main\res\"

# mdpi (48×48)
Copy-Item "panda_48.png" "$projectPath\mipmap-mdpi\ic_launcher.png"
Copy-Item "panda_48.png" "$projectPath\mipmap-mdpi\ic_launcher_round.png"

# hdpi (72×72)
Copy-Item "panda_72.png" "$projectPath\mipmap-hdpi\ic_launcher.png"
Copy-Item "panda_72.png" "$projectPath\mipmap-hdpi\ic_launcher_round.png"

# xhdpi (96×96)
Copy-Item "panda_96.png" "$projectPath\mipmap-xhdpi\ic_launcher.png"
Copy-Item "panda_96.png" "$projectPath\mipmap-xhdpi\ic_launcher_round.png"

# xxhdpi (144×144)
Copy-Item "panda_144.png" "$projectPath\mipmap-xxhdpi\ic_launcher.png"
Copy-Item "panda_144.png" "$projectPath\mipmap-xxhdpi\ic_launcher_round.png"

# xxxhdpi (192×192)
Copy-Item "panda_192.png" "$projectPath\mipmap-xxxhdpi\ic_launcher.png"
Copy-Item "panda_192.png" "$projectPath\mipmap-xxxhdpi\ic_launcher_round.png"

Write-Host "✅ 图标复制完成！" -ForegroundColor Green
```

## ❓ 常见问题

### Q: 图片失真了怎么办？
A: 使用更高的分辨率原图（1024×1024），然后缩小

### Q: 背景不是透明的怎么办？
A: 使用支持透明背景的 PNG 格式

### Q: 圆形图标被裁剪了怎么办？
A: 确保图标内容居中，留出足够的边距

### Q: 只有 PNG 格式可以用吗？
A: 也可以用 WebP 格式，但 PNG 兼容性最好
