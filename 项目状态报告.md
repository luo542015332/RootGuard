## 📌 重要提示

### 图标设置说明

由于用户无法在线上传图标，提供了本地处理方案：

1. 使用 Windows 画图工具生成多个尺寸的图标
2. 使用 `copy-icons.ps1` 批量复制到项目
3. 运行 `.\gradlew.bat clean` 清理缓存
4. 重新构建并安装

详细步骤请参考：**图标设置详细步骤.md**
