# PandaSU 问题修复方案

## 问题1：SQLite3 依赖问题

### 问题分析
- SQLite3 命令在设备上不可用或无法访问
- 影响 Magisk Hide 策略配置

### 修复方案
1. **检查 SQLite3 可用性**：在执行 SQLite3 命令前先检查命令是否存在
2. **提供备选方案**：如果 SQLite3 不可用，使用 Magisk 原生命令
3. **错误处理优化**：提供更详细的错误信息和备选方案

### 具体修改
在 `RootHider.kt` 中修改 `disableRootForPackage` 函数：
```kotlin
private suspend fun disableRootForPackage(packageName: String): Boolean = withContext(Dispatchers.IO) {
    try {
        // 先检查 sqlite3 是否可用
        val sqlite3Available = runSuCommand("which sqlite3 2>/dev/null")
        
        if (sqlite3Available) {
            runSuCommand("sqlite3 /data/adb/magisk.db \"INSERT OR REPLACE INTO policies (package_name, policy) VALUES ('$packageName', 1)\"")
        } else {
            // 使用 Magisk 原生命令
            runSuCommand("magisk --denylist add $packageName")
        }
        true
    } catch (e: Exception) { 
        Logger.e("disableRootForPackage failed: ${e.message}")
        false 
    }
}
```

## 问题2：mount 命令权限问题

### 问题分析
- `mount --bind` 命令执行失败（exitCode=1）
- 可能原因：权限不足、路径不存在或系统限制

### 修复方案
1. **路径验证**：在执行 mount 前验证目标路径是否存在
2. **权限检查**：检查 mount 命令的可用性
3. **备选方案**：如果 mount 失败，使用其他隐藏方法
4. **错误处理**：提供更详细的错误信息

### 具体修改
在 `RootHider.kt` 中创建辅助函数：
```kotlin
private suspend fun safeMountBind(source: String, target: String): Boolean = withContext(Dispatchers.IO) {
    try {
        // 1. 检查目标路径是否存在
        val targetExists = runSuCommandOutput("ls $target 2>/dev/null").isNotEmpty()
        if (!targetExists) {
            Logger.d("safeMountBind: Target path $target does not exist")
            return@withContext false
        }
        
        // 2. 检查 mount 命令是否可用
        val mountAvailable = runSuCommand("which mount 2>/dev/null")
        if (!mountAvailable) {
            Logger.d("safeMountBind: mount command not available")
            return@withContext false
        }
        
        // 3. 执行 mount --bind
        val result = runSuCommand("mount --bind $source $target 2>/dev/null")
        if (!result) {
            Logger.d("safeMountBind: mount --bind failed for $source -> $target")
        }
        result
    } catch (e: Exception) {
        Logger.e("safeMountBind exception: ${e.message}")
        false
    }
}
```

1. 多层级检测策略
