# 安装页面功能修复总结

## 📋 问题描述
用户反馈截图显示安装页面中的选项点击没有反应：
- "选择一个文件"（建议选择 init_boot 分区镜像）
- "直接安装（推荐）"（自动修补并安装）
- "安装到未使用的槽位（OTA 后）"（适用于 A/B 分区设备）

## 🔍 根本原因分析

### 1. ViewModel 功能未实现
- `InstallViewModel.proceedInstallation()` 只有 TODO 注释
- 缺少实际的安装逻辑实现

### 2. 缺少文件选择功能
- 没有文件选择器实现
- 无法选择 init_boot.img 或 boot.img 文件

### 3. 无状态反馈机制
- 没有安装进度显示
- 没有错误信息反馈
- 没有安装结果提示

## ✅ 修复方案

### 1. 完善 InstallViewModel.kt
```kotlin
// 添加安装状态枚举
enum class InstallStatus {
    IDLE, SELECTING_FILE, INSTALLING, SUCCESS, ERROR
}

// 实现安装逻辑
suspend fun installKernelSU(): Boolean {
    when (installMethod) {
        InstallMethod.SELECT_FILE -> installFromFile(selectedFile!!)
        InstallMethod.DIRECT -> installDirectly()
        InstallMethod.OTHER_SLOT -> installToOtherSlot()
    }
}
```

### 2. 添加文件选择功能
- 使用 `ActivityResultContracts.GetContent()` 打开文件选择器
- 自动将文件复制到临时目录
- 验证文件类型（.img 文件）
- 显示选中的文件路径

### 3. 实现三种安装方式
1. **选择一个文件**：`ksud -o install <file_path>`
2. **直接安装**：`ksud -o install`（自动安装）
3. **安装到未使用槽位**：检测当前槽位，安装到另一槽位

### 4. 添加状态反馈
- 显示选中的文件信息
- 显示安装进度条
- 显示安装日志
- 显示错误信息
- 提供错误关闭按钮

## 📱 用户操作流程

### 操作步骤：
1. **进入安装页面**
2. **选择安装方式**（三选一）
   - 选择一个文件 → 自动打开文件选择器
   - 直接安装 → 直接进入下一步
   - 安装到未使用槽位 → 检测槽位后安装
3. **点击"下一步"按钮**
4. **查看安装进度和结果**

### 自动触发：
- 当选择"选择一个文件"时，自动打开文件选择器
- 选择文件后自动显示文件路径
- 安装过程中自动显示进度
- 安装完成自动显示结果

## 🔧 技术实现细节

### 文件选择器
```kotlin
val filePickerLauncher = rememberLauncherForActivityResult(
    contract = ActivityResultContracts.GetContent()
) { uri: Uri? ->
    uri?.let {
        viewModel.selectFileFromUri(context, it)
    }
}
```

### 安装命令
```kotlin
// 从文件安装
Runtime.getRuntime().exec(arrayOf("su", "-c", "ksud", "-o", "install", filePath))

// 直接安装
Runtime.getRuntime().exec(arrayOf("su", "-c", "ksud", "-o", "install"))

// 安装到指定槽位
Runtime.getRuntime().exec(arrayOf("su", "-c", "ksud", "-o", "install", "--slot", targetSlot))
```

### 槽位检测
```kotlin
val slotCheckProcess = Runtime.getRuntime().exec(arrayOf("su", "-c", "getprop", "ro.boot.slot_suffix"))
val currentSlot = slotCheckProcess.inputStream.bufferedReader().readText().trim()
val targetSlot = if (currentSlot == "_a") "_b" else "_a"
```
