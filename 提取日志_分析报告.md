### A. 日志提取脚本
```bash
# Windows PowerShell
Get-ChildItem *.txt | Where-Object {$_.Name -match "log|crash|anr|error"} | 
  ForEach-Object {
    $content = Get-Content $_.FullName -First 20
    Write-Output "=== $($_.Name) ==="
    Write-Output $content
    Write-Output ""
  }
```
