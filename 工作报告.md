## 一、项目概述

PandaTurbo 是一款**免Root游戏优化工具**，通过 Android 系统提供的无障碍服务、UsageStatsManager 等接口实现游戏检测、后台清理、显示优化等功能。

### 核心功能
- 游戏自动检测与优化
- 一键清理后台进程
- 显示/网络优化（基于官方 Game Mode API）
- 悬浮窗性能监控
- 无障碍服务自动化操作

### 3.2 GameManager.kt
**路径**: `app/src/main/java/com/pandasu/turbo/GameManager.kt`

**功能**: 核心优化引擎，提供所有优化方法

**主要方法**:
| 方法名 | 功能 | 实现方式 |
|--------|------|----------|
| `oneKeyTurbo()` | 一键Turbo | 调用所有优化项 |
| `cleanBackgroundApps()` | 清理后台 | UsageStatsManager + am force-stop |
| `optimizeDisplay()` | 显示优化 | GameMode API (Android 12+) |
| `optimizeNetwork()` | 网络优化 | 调整网络优先级 |

---

### 3.3 GameAccessibilityService.kt
**路径**: `app/src/main/java/com/pandasu/turbo/GameAccessibilityService.kt`

**功能**: 无障碍服务，检测前台应用变化，自动触发优化

**工作原理**:
1. 监听 `TYPE_WINDOW_STATE_CHANGED` 事件
2. 获取当前前台应用包名
3. 与游戏列表匹配
4. 匹配成功则自动执行优化

---

**路径**: `app/src/main/java/com/pandasu/turbo/GameDetectionService.kt`

**支持的检测方式**:
- UsageStatsManager 查询前台应用

## 八、技术限制

### 免Root 方案的限制
1. **无法真正修改温控** - 只能通过显示数据误导系统感知
2. **无法超频 CPU/GPU** - 需要内核权限
3. **无法深度清理** - 只能清理部分后台
4. **Game Mode API** - 仅 Android 12+ 支持

### 无障碍服务限制
- 需要用户手动开启
- 部分厂商可能拦截
- 功耗较高
